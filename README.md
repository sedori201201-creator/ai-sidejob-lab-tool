# AI副業ラボ 投稿メーカー

note記事のテーマから、本文、有料部分、サムネ生成プロンプト、投稿チェックリストを作る静的Webツールです。

## 使い方

`index.html` をブラウザで開きます。

入力するもの:

- 記事テーマ
- 読者
- 読者のゴール
- 有料/無料
- 価格
- サムネの雰囲気
- ツール内アフィリエイト枠

出力されるもの:

- note本文
- 有料部分
- サムネ生成プロンプト
- 投稿チェックリスト

## アフィリエイト枠

画面上部にPRカードを表示できます。記事本文には入れず、ツール画面内だけに表示します。

設定できるもの:

- リンクURL
- 表示タイトル
- 説明文
- ボタン文言

## ファイル

- `index.html`: 画面
- `styles.css`: 見た目
- `app.js`: 生成ロジック
