# AI副業ラボ 投稿メーカー マニュアル

## このツールでできること

`AI副業ラボ 投稿メーカー` は、note記事を作るための下書き支援ツールです。

テーマを入力すると、以下をまとめて作れます。

- note記事タイトル
- 本文
- 有料部分
- サムネ生成プロンプト
- 投稿チェックリスト
- ツール内アフィリエイトPRカード

記事本文にアフィリエイトリンクを自動挿入するのではなく、ツール画面の上部にPRカードとして表示できます。

## 起動方法

以下のファイルをブラウザで開きます。

```text
ai-sidejob-post-maker/index.html
```

ローカルで動く静的Webツールなので、サーバー起動は不要です。

## 基本の使い方

まずは左側の入力欄を上から順番に埋めます。

最低限入力するものは以下の5つです。

1. 記事テーマ
2. 読者
3. 読者のゴール
4. 有料 / 無料
5. サムネの雰囲気

アフィリエイトを使う場合だけ、下の `ツール内アフィリエイト枠` も入力します。

全部入れたら、最後に `投稿セットを作る` を押します。

すると右側に、以下が自動で出ます。

- 本文
- サムネ生成プロンプト
- 投稿チェックリスト

### 1. 記事テーマを入力する

例:

```text
アプリと課金システムで月5万円を作る方法
```

ここに入れた内容をもとに、タイトル・本文・有料部分が作られます。

### 2. 読者を入力する

例:

```text
AI副業をこれから始めたい初心者
```

読者を具体的にすると、本文の方向性がブレにくくなります。

### 3. 読者のゴールを入力する

例:

```text
月5万円の収入導線を作る
```

読者が記事を読んだあとにどうなってほしいかを入れます。

### 4. 文体を選ぶ

選べる文体:

- やさしく実践的
- 短く力強く
- 有料note向けに濃く

有料noteにする場合は、`有料note向けに濃く` が向いています。

### 5. 価格を入力する

有料noteにする場合の価格です。

例:

```text
500
```

本文中の有料部分の説明にも反映されます。

### 6. 販売形式を選ぶ

選択肢:

- 有料note
- 無料note

有料noteを選ぶと、本文に有料部分の案内が入ります。

### 7. サムネの雰囲気を入力する

例:

```text
黄色背景に黒文字、シンプル、正円にも使える
```

この内容をもとに、サムネ生成プロンプトが作られます。

## 投稿セットを作る

入力が終わったら、以下のボタンを押します。

```text
投稿セットを作る
```

右側に3つの出力が表示されます。

- 本文
- サムネ
- 投稿

## 本文タブ

noteに貼り付ける記事本文です。

使い方:

1. `本文` タブを開く
2. `コピー` を押す
3. noteの編集画面に貼り付ける

## サムネタブ

画像生成AIに渡すためのサムネ生成プロンプトです。

使い方:

1. `サムネ` タブを開く
2. `コピー` を押す
3. 画像生成AIに貼り付ける
4. 生成された画像をnoteのサムネに設定する

## 投稿タブ

note公開前のチェックリストです。

確認すること:

- タイトル
- 販売形式
- 価格
- サムネ設定
- 有料部分の境界
- 本文の崩れ
- PR表記とリンク先

## Markdown保存

`Markdown保存` を押すと、本文を `.md` ファイルとして保存できます。

保存されるファイル名:

```text
note-post.md
```

## 全部コピー

`全部コピー` を押すと、以下をまとめてコピーできます。

- 本文
- サムネ生成プロンプト
- 投稿チェックリスト

## アフィリエイト枠の使い方

このツールでは、記事本文の中ではなく、ツール画面の上部にアフィリエイト用PRカードを表示できます。

### 1. 表示をONにする

`ツール内アフィリエイト枠` の `表示` にチェックを入れます。

### 2. リンクURLを入力する

例:

```text
https://example.com/affiliate
```

URLが入っていない場合、PRカードは表示されません。

### 3. 表示タイトルを入力する

例:

```text
おすすめAIツール
```

### 4. ボタン文言を入力する

例:

```text
詳しく見る
```

### 5. 説明文を入力する

例:

```text
AI副業の記事作成や作業効率化に役立つツールです。
```

## アフィリエイト枠の注意点

アフィリエイトリンクを使う場合は、PRであることが分かる表示を入れてください。

このツールでは、PRカードに `PR` ラベルが表示されます。

リンクには以下の属性が入っています。

```html
rel="sponsored noopener"
```

## おすすめ運用

### 無料noteの場合

本文は短めにして、最後に次の記事やサービスへ誘導します。

おすすめ設定:

- 販売形式: 無料note
- 文体: やさしく実践的
- サムネ: 黄色背景に黒文字

### 有料noteの場合

無料部分で全体像を見せ、有料部分にテンプレートや手順を入れます。

おすすめ設定:

- 販売形式: 有料note
- 価格: 500円
- 文体: 有料note向けに濃く
- 有料部分: テンプレート、チェックリスト、プロンプト集

## note公開までの流れ

1. このツールで本文を作る
2. 本文をコピーする
3. note編集画面に貼り付ける
4. サムネ生成プロンプトで画像を作る
5. サムネ画像をnoteに設定する
6. 有料noteなら価格と有料エリアを設定する
7. 投稿チェックリストを確認する
8. 問題なければ公開する
